## 项目概述

「浮生」是一个单页工具网站，解决"无聊时不知道做什么"的问题。
两种方式获取活动建议：随机抽取，以及基于五行运势的匹配推荐。

## 设计原则

- **克制**：单页，不翻页，不跳转。所有交互在一屏内完成。
- **自然**：颜色、推荐逻辑与当日干支挂钩，页面每天有微妙变化，不刻意。
- **典雅简洁**：文案有古风韵味但不啰嗦，一行说清的绝不用两行。不说"为你推荐""智能匹配"这类AI套话。

## 技术选型

- 纯静态页面：HTML + CSS + 原生 JS，无框架，无构建工具
- 所有逻辑在客户端完成，无需后端
- 干支计算用 JS 实现，不依赖外部 API
- 字体通过 Google Fonts 加载 思源宋体（Noto Serif SC）

## 核心功能

### 1. 随机抽取

页面中央一个醒目的按钮，点击后从活动池中随机抽取一条展示。
抽取时有翻牌或滚动动画，避免生硬切换。
活动池约 50-100 条，按大类分组，随机时不区分类别。

### 2. 五行运势匹配

用户输入出生月日后，五行运势匹配自动融入随机逻辑：
- 用户月支对应的五行属性，作为"本命五行"
- 当日天干对应的五行属性
- 当日五行对本命五行的生克关系（生我/同我/我生/我克/克我）

每条活动预设 1-2 个五行标签（如"利火""利水"），匹配时优先推荐与当日运势相合的活动：
- 当日生本命五行 → 推荐该五行标签的活动
- 当日本命同五行 → 推荐同五行标签的活动
- 其他情况 → 随机抽取
- 用户未输入生日时，纯随机抽取

干支计算原理：
- 以 1900-01-01（甲戌日，干支序数 11）为基准
- 日干支序数 = (距基准日天数 + 11) mod 60
- 天干 = 日干支序数 mod 10，地支 = 日干支序数 mod 12
- 天干五行映射：甲乙=木，丙丁=火，戊己=土，庚辛=金，壬癸=水

### 3. 当日运势简述

基于当日日柱显示一句话描述，例如"丙午日，火旺，做事利落"。

### 4. 活动历史记录

页面左栏展示最近 5 条抽取过的活动，供回顾。超过 5 条的旧记录自动挤掉。

## 数据结构

```
活动对象:
{
  id: number,
  text: string,        // "去公园散步二十分钟"
  tags: string[],      // ["利火", "利木"]
  category: string     // 动手 | 出门 | 静心 | 社交 | 随便
}

历史记录:
[activityId1, activityId2, ...]  // 最多 5 条，localStorage 存储
```

## 配色方案

暖色调基底，根据当日天干五行切换同色系流动渐变。

底色：暖米色基底（#FBF7F0），叠加轻微的同色系流动渐变，用 CSS animation 缓慢推移，视觉上像水墨在纸上微微晕开。

各五行强调色渐变（用于按钮、框边、高亮文字）：

| 日干五行 | 渐变区间                         | 方向     |
|----------|----------------------------------|----------|
| 木       | #8B9A46 → #A3B05A → #6B7A36     | 对角线   |
| 火       | #C75B39 → #D9805A → #A84D2B     | 对角线   |
| 土       | #C4953A → #D4A54A → #A4752A     | 对角线   |
| 金       | #C4A882 → #D4B892 → #A48862     | 对角线   |
| 水       | #7B8FA1 → #96AAB5 → #617585     | 对角线   |

**渐变实现要点**：
- 用 CSS `linear-gradient` 同色系多 stop，非跨色相跳变
- 背景层用 `background-size: 200% 200%` + 慢速 `@keyframes` 推移（周期 20-30s），产生类似水墨流动的微弱动感
- 按钮、框体用静态渐变叠加，不参与动画，避免眼花
- 日切换在每天 0 点自动切换色系，渐变区间随之更新

## 字体

- 正文：思源宋体 / Noto Serif SC（Google Fonts 加载）
- 回退：Songti SC → SimSun → serif
- 标题字号 28-36px，正文 16-18px，行距 1.8
- 不用细体、不用斜体

## 页面布局

```
┌──────────────────────────────────────────────────────────┐
│  浮生                                   请输入出生月日    │
│  （大字·左上角）                        （小字·右上角）  │
│                                                          │
│  ┌──────────┐  ┌──────────────┐  ┌──────────────┐       │
│  │          │  │              │  │              │       │
│  │ 历史记录  │  │ 当日干支      │  │ 今日推荐      │     │
│  │          │  │ + 运势简述    │  │ 做的事        │     │
│  │（最近5条）│  │              │  │              │       │
│  │          │  │              │  │              │       │
│  └──────────┘  └──────────────┘  └──────────────┘       │
│                ┌──────────────────────────────────┐      │
│                │ [ 随机一下 ]   │  翻牌/动画区域   │      │
│                │（宽=输入框宽）  │                  │      │
│                └──────────────────────────────────┘      │
└──────────────────────────────────────────────────────────┘
```

**分区说明**：
- 左栏：历史记录，「浮生」正下方
- 右上：生日输入框（小字，淡色 placeholder "请输入出生月日"）
- 右中（输入框下方）：两个等大中号框并排，偏正方形比例
  - 左框：当日干支 + 运势简述
  - 右框：当日推荐做的事
  - 两框颜色：网页底色高不透明度，与背景融而不糊，微弱区分
- 右下：一个宽框，左边界与右中左框对齐，右边界与右中右框对齐
  - 左侧：随机按钮，宽度与生日输入框一致，按钮内淡色「随机一下」；按钮边框/底色用当日渐变
  - 右侧：翻牌或滚动动画展示区
- 整体横屏比例，内容垂直居中，不强制滚动
